# Marketing-Campaign-Performance-A-B-Analysis

## Overview

This project analyzes multi-channel marketing campaign performance using data from Facebook and Google Ads (AdWords). The objective is to evaluate campaign effectiveness, customer engagement, and ROI, and identify opportunities to optimize marketing strategies using data-driven insights.

## Objectives
- Analyze campaign performance across multiple channels
- Track and evaluate key marketing KPIs
- Perform A/B testing to compare campaign effectiveness
- Identify trends, patterns, and conversion drivers
- Generate actionable insights to improve ROI and targeting
## Dataset
- Duration: 365 days (daily campaign data)
- Channels: Facebook Ads & Google AdWords
## Key features:
- Ad Views
- Clicks
- Conversions
- Cost per Ad
- Click-Through Rate (CTR)
- Conversion Rate
- Cost per Click (CPC)
## Tools & Technologies
- Python (Pandas, NumPy)
- Visualization: Matplotlib, Seaborn
- Statistical Analysis: SciPy
- Machine Learning (basic): Scikit-learn
## Key Analysis Performed
- Exploratory Data Analysis (EDA) on campaign performance
- KPI Tracking: CTR, Conversion Rate, CPC
- A/B Testing: Facebook vs AdWords campaign comparison
- Trend and distribution analysis of clicks and conversions
- Conversion segmentation (low, medium, high performance)
- Correlation analysis between clicks and conversions
- Time-based analysis for performance patterns
## Key Insights
- Identified performance differences between channels, highlighting stronger conversion efficiency in specific campaigns
- Found a positive relationship between clicks and conversions, validating campaign engagement impact
- Segmentation revealed high-performing conversion windows, enabling better targeting strategies
- Noticed variability in campaign performance, suggesting opportunities for budget optimization and strategy refinement
## Business Impact
- Improved understanding of campaign effectiveness and ROI drivers
- Enabled data-driven decision-making for marketing strategy
- Provided recommendations for channel optimization and budget allocation
- Supported targeted campaign improvements through segmentation and analysis
## Future Improvements
- Incorporate customer-level data for deeper segmentation
- Perform advanced attribution modeling
- Implement predictive modeling for conversion forecasting
- Build an interactive dashboard (Power BI / Tableau)
## Conclusion
This project demonstrates how data analytics and A/B testing can be leveraged to evaluate marketing campaigns, optimize performance, and drive better ROI and customer engagement.
